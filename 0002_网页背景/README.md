# p5js实现简单的网页背景图

## 说明: 

可用于博客等网页的背景

## 使用: 

本目录public包结构为

- public
  + assets --- p5js依赖
  + js --- 实现效果的js文件
  + index.html --- 网页

使用时拷贝assets文件夹, js文件夹, html并保证路径正确

若不想使用本地预设即assets文件夹中内容, 可以使用p5js在CDN上的文件.

```html
<script src="https://cdnjs.cloudflare.com/ajax/libs/p5.js/0.9.0/p5.js"></script>
```
